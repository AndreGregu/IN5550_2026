2: Evaluation 2_draft:
	Improvements: 
	Added metadata features in addition to the text
	Original model only used "text" as a feature with TF-IDF в†’ Ridge regression
	From this i addded some extra structures such as: 
		- text_length
		- word_count
		- is_words
		- collection_phase
	the model then uses TF-IDF(text) + metadata в†’ Ridge regression
	text_length and word_count helps because the strucutre of essays are different from short-emotion words. 
	is_word tells the model whether the entry is a full essay or a feeling-word-list which matters because those two text types are explicitly different in the task. 
	collection_phase provides some weak temporal/contextual information

	Performance: 	
	r_between 0.734 (1.31e-16) 
	r_within 0.453 (7.10e-09) 
	r_composite 0.612 
	mae_between 0.422 
	mae_composite 0.669 
	mae_within 0.823 

	Key differences: 
	r_composite: 0.597 в†’ 0.612
	r_between: 0.714 в†’ 0.734
	r_within: 0.448 в†’ 0.453 (barely noticable)

	Interpretation: 
	Better capture global patterns (between users)
	But still not much better at within-user variation over time

	Bottleneck: 
	modeling user-specific dynamics
